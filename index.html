MOCPy is a Python library allowing easy creation, parsing and manipulation of MOCs (Multi-Order Coverage maps). As of the version 0.6.0, MOCPy is only distributed for Python 3.5/3.6 and 3.7. Python 2 users can still refer to the 0.5.7 version. Its code is hosted on GitHub and distributed under the BSD-3 license.

What is a MOC ?¶

MOC is an IVOA standard enabling description of arbitrary sky regions. Based on the HEALPix sky tessellation, it maps regions on the sky into hierarchically grouped predefined cells.

MOCPy provides the MOC and TimeMOC classes handling respectively the manipulation of spatial and temporal MOCs.
